# python-basic-scripts
collection of simple python scripts
1. Rock Paper Scissors Game
A command‑line game where the user plays Rock‑Paper‑Scissors against the computer.
Skills demonstrated:
- Random number generation
- Conditional logic
- User input handling

2. Quiz Game
A simple quiz program that asks multiple‑choice or true/false questions and tracks the user’s score.
Skills demonstrated:
- Lists and dictionaries
- Loops
- Score tracking
- Basic validation

3. Shopping Cart Program
A basic shopping cart simulation where users can add items, view their cart, and calculate totals.
Skills demonstrated:
- Lists
- Loops
- Basic arithmetic
- Menu‑driven programs
